## Sobre

Este projeto tem como objetivo oferecer uma aplicação de gerenciamento de tarefas simples e eficiente, facilitando o controle dos afazeres do dia a dia. Ideal para praticar Node.js, manipulação de rotas, middleware e conceitos fundamentais do desenvolvimento web.

## Funcionalidades

- Adicionar novas tarefas
- Listar todas as tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Editar descrições de tarefas

*Obs.: personalize esta seção de acordo com as funcionalidades reais implementadas no código.*

## Tecnologias Utilizadas

- [Node.js](https://nodejs.org/)
- [Express.js](https://expressjs.com/) (caso utilize)
- [HTML](https://developer.mozilla.org/pt-BR/docs/Web/HTML)
- Outras dependências listadas no `package.json`

## Instalação

1. Clone este repositório:
   ```bash
   git clone https://github.com/gabrieleaamorim/to-do-nodejs.git
   cd to-do-nodejs
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor:
   ```bash
   npm start
   ```
   *Verifique o arquivo package.json para o script de inicialização correto.*

## Como Usar

Após rodar a aplicação, acesse no seu navegador:
```
http://localhost:3000
```
Utilize a interface web ou as rotas da API (caso disponíveis) para gerenciar suas tarefas.
